# FitForge 🏋️ — Forge Your Best Self

A premium, full-featured fitness tracking SaaS application built with React + Vite.

## Features
- 🔐 Authentication (Login / Register)
- 📊 Dashboard with charts & stats
- 💪 Workout Tracker with exercise library
- 🥗 Nutrition & Macro Tracker
- 💧 Water intake tracker
- 📈 Body Progress & Measurements
- 🎯 Goals & Achievement Badges
- 📉 Analytics with heatmap
- 🤖 AI Fitness Coach Chatbot
- 👤 Profile Management
- 🛡️ Admin Panel

## Quick Start

### 1. Install dependencies
\`\`\`bash
npm install
\`\`\`

### 2. Run the development server
\`\`\`bash
npm run dev
\`\`\`

### 3. Open in browser
Visit: http://localhost:5173
